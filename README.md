# green-regex

## Description

This project was to create a gist for a regex. In this example its for a regex to check for valid URL. This gist will show a break down of the characters used in this expression. 

`/^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$/`
